ALL FRONTEND TOPICS

THE TOPICS ARE :

  • HTML

    • HTML (HyperText Markup Language) is the code that is used to structure a web page and its content.

  • CSS

    • CSS describes how HTML elements are to be displayed on screen, paper, or in other media. CSS saves a lot of work. It can control the layout of multiple web pages all at once.

  • JAVASCRIPT

    • JavaScript are image manipulation, form validation, and dynamic changes of content. To select an HTML element, JavaScript most often uses the document.getElementById() method.

  • TS
    •

    TypeScript is a syntactic superset of JavaScript which adds static typing. This basically means that TypeScript adds syntax on top of JavaScript, allowing developers to add types. TypeScript being a "Syntactic Superset" means that it shares the same base syntax as JavaScript, but adds something to it.
